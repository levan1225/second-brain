For Slack + Outlook: use Claude Desktop

Claude Desktop already ships with Slack and Outlook connectors that handle OAuth for you. Set them up in Claude Desktop → Settings → Connectors, then Claude sessions can post to Slack or read your calendar — alongside the secondbrain MCP tools.

Example: ask Claude "Post today's morning briefing to my Slack DM". Claude reads the briefing file with secondbrain's read_wiki_page tool, then sends via Claude's own Slack connector.
